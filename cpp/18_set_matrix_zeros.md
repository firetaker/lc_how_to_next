# 矩阵置零（LeetCode #73）

## 题目描述

> 若元素为 0，则将整行和整列置零。要求 O(mn) 时间，O(1) 空间。

**示例：**
- 输入：`[[1,1,1],[1,0,1],[1,1,1]]` → 输出：`[[1,0,1],[0,0,0],[1,0,1]]`

---

## 梯度分析

### 🔵 第一层：额外数组记录 O(m+n) 空间

用 `row_set` 和 `col_set` 记录哪些行和列需要置零。空间 O(m+n)，不符合 O(1)。

---

### 🟢 第二层：利用自身首行首列做标记 O(1) 空间

**核心洞察：** 若 `matrix[i][j] == 0`，就把 `matrix[i][0]` 和 `matrix[0][j]` 标记为 0——利用矩阵的首行首列代替额外的标记数组。

**三个步骤：**

```python
# 1. 单独检查首行首列是否有 0（因为会被后续操作覆盖）
row_zero = any(matrix[0][j] == 0 for j in range(n))
col_zero = any(matrix[i][0] == 0 for i in range(m))

# 2. 第一遍扫描：用首行首列做标记
for i in range(1, m):
    for j in range(1, n):
        if matrix[i][j] == 0:
            matrix[i][0] = 0   # 标记第 i 行
            matrix[0][j] = 0   # 标记第 j 列

# 3. 第二遍：根据标记置零（跳过首行首列）
for i in range(1, m):
    if matrix[i][0] == 0:
        matrix[i][:] = [0] * n
for j in range(1, n):
    if matrix[0][j] == 0:
        for i in range(m):
            matrix[i][j] = 0

# 4. 处理首行首列
if row_zero: matrix[0][:] = [0] * n
if col_zero:
    for i in range(m): matrix[i][0] = 0
```

**为什么这样可行？**
- 首行首列的元素被标记后，在第二遍可以用来决定对应行/列是否置零
- 但首行首列自身的置零需要单独判断（因为第一遍扫描跳过了它们）

**图解：**

```
原始:
[1, 1, 1]
[1, 0, 1]
[1, 1, 1]

Step 1: 检查首行首列 → 无0

Step 2: 扫描内部 (i=1..1, j=1..1):
  matrix[1][1]==0 → matrix[1][0]=0, matrix[0][1]=0

Step 3: 置零
  matrix[1][0]==0 → 第1行全0
  matrix[0][1]==0 → 第1列全0

Step 4: 首行首列 → 无需置零

结果:
[1, 0, 1]
[0, 0, 0]
[1, 0, 1] ✅
```

**复杂度：**
- 时间：O(mn)
- 空间：O(1)

---

## 破题总结

> **用矩阵自身首行首列做标记数组：扫描时若 matrix[i][j]==0 则标记 matrix[i][0] 和 matrix[0][j] 为 0；第二遍用这些标记决定行列置零；首行首列单独处理。**
