# 相交链表（LeetCode #160）

## 题目描述

> 判断两链表是否相交，返回相交的第一个节点。

**示例：**
- A: `4→1→[8→4→5]`；B: `5→6→1→[8→4→5]` → 相交节点：`[8→4→5]`

---

## 梯度分析

### 🔵 第一层：哈希集合记录节点

遍历 A 存入 set，再遍历 B 找第一个在 set 中的节点。O(n+m) 时间，O(n) 空间。

---

### 🟢 第二层：双指针切换法 O(n+m) / O(1)

```cpp
ListNode *a = headA, *b = headB;
while (a != b) {
    a = a ? a->next : headB;
    b = b ? b->next : headA;
}
return a;
```

**数学：a 走 A+B，b 走 B+A，尾部等长消除后必在相交点相遇。**

**复杂度：**
- 时间：O(n+m)
- 空间：O(1)

---

## 破题总结

> **双指针走完各自链表后切换到另一条继续走——尾部等长消除后，两者走过的总距离相等，必然在相交点或 nil 处相遇。**
