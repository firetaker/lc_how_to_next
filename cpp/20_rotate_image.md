# 旋转图像（LeetCode #48）

## 题目描述

> 顺时针旋转 n×n 矩阵 90 度，原地修改。

**示例：**
- 输入：`|1,2,3|` → 输出：`|7,4,1|`
         `|4,5,6|`         `|8,5,2|`
         `|7,8,9|`         `|9,6,3|`

---

## 梯度分析

### 🔵 第一层：另存旋转后坐标 O(n²)

手动计算旋转后坐标 `(i,j) → (j, n-1-i)`，另开数组保存。

---

### 🟢 第二层：两次翻转 O(1)

**核心洞察：** 顺时针 90° 旋转 = **沿主对角线翻转** + **沿水平中线翻转**

```cpp
int n = matrix.size();
// Step 1: 转置（主对角线翻转）
for (int i = 0; i < n; ++i)
    for (int j = i + 1; j < n; ++j)
        swap(matrix[i][j], matrix[j][i]);
// Step 2: 每行 reverse（水平翻转）
for (int i = 0; i < n; ++i)
    reverse(matrix[i].begin(), matrix[i].end());
```

**验证：**

```
原始:       Step1(转置):    Step2(每行reverse):
[1,2,3]     [1,4,7]        [7,4,1]
[4,5,6]  →  [2,5,8]    →   [8,5,2]
[7,8,9]     [3,6,9]        [9,6,3] ✅
```

**复杂度：**
- 时间：O(n²)
- 空间：O(1)

---

## 破题总结

> **顺时针90度旋转 = 转置（主对角线翻转）+ 每行 reverse（水平翻转）。两步各 O(n²)，无需额外空间。**
