# 盛最多水的容器（LeetCode #11）

## 题目描述

> 两线 `i` 和 `j` 之间水量 = `min(height[i], height[j]) × (j - i)`，求最大。

**示例：**
- 输入：`[1, 8, 6, 2, 5, 4, 8, 3, 7]`
- 输出：`49`

---

## 梯度分析

### 🔵 第一层：暴力枚举 O(n²)

枚举所有两两组合，计算面积。数据量大的情况下不可接受。

---

### 🟢 第二层：相向双指针 O(n)

**核心洞察：** 面积由较短的木板决定。固定长板后，只有找到更高的短板才可能增大面积。

```cpp
int left = 0, right = height.size() - 1;
int max_water = 0;
while (left < right) {
    int h = min(height[left], height[right]);
    int w = right - left;
    max_water = max(max_water, h * w);
    if (height[left] < height[right])
        ++left;   // 移动短板
    else
        --right;  // 移动短板
}
```

**为什么移动短板？**
- 若移动长板：新高度 ≤ 原来的矮板，且宽度减小 → 面积一定不增加
- 移动短板：新高度可能增加 → 有可能获得更大面积

**复杂度：**
- 时间：O(n)，指针最多移动 n 次
- 空间：O(1)

---

## 破题总结

> **相向双指针中，每次移动较短边的指针——面积由短板决定，固定长板后只有找到更高短板才可能增大面积，指针移动次数严格限制在 O(n)。**
