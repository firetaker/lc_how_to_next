# 移动零（LeetCode #283）

## 题目描述

> 将数组中所有 `0` 移动到末尾，保持非零元素相对顺序，原地修改。

**示例：**
- 输入：`[0, 1, 0, 3, 12]` → 输出：`[1, 3, 12, 0, 0]`

---

## 梯度分析

### 🔵 第一层：创建新数组（不符合 O(1) 空间要求）

收集非零元素，再补零。空间 O(n)，不符合原地要求。

---

### 🟢 第二层：双指针原地交换

`left` 维护「已处理非零元素」的边界，`right` 扫描数组：

```cpp
int left = 0;
for (int right = 0; right < nums.size(); ++right) {
    if (nums[right] != 0) {
        swap(nums[left], nums[right]);
        ++left;
    }
}
```

**图解 `[0, 1, 0, 3, 12]`：**

| 步骤 | right | 动作 | nums |
|------|-------|------|------|
| 初始 | - | - | [0, 1, 0, 3, 12] |
| right=0 | 跳过（值为0） | - | |
| right=1 | 交换 [1,0] → [1,0,0,3,12] | left=1 | |
| right=2 | 跳过 | - | |
| right=3 | 交换 [3,0] → [1,3,0,0,12] | left=2 | |
| right=4 | 交换 [12,0] → [1,3,12,0,0] | left=3 | |

**复杂度：**
- 时间：O(n)，每个元素最多交换一次
- 空间：O(1)

---

## 破题总结

> **双指针中，`left` 维护「已处理非零元素」的边界，每当 `right` 遇到非零就与 `left` 交换并右移——类似「把非零往前挤」。**
