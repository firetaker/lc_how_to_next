# 搜索二维矩阵（LeetCode #74）

## 题目描述

> 给定 m×n 矩阵，每行升序，每行第一个元素大于上一行最后一个元素。判断 target 是否存在。

**示例：**
- 输入：`[[1,3,5,7],[10,11,16,20],[23,30,34,60]], target=3` → 输出：`True`

---

## 梯度分析

### 🔵 第一层：二维二分 O(log m + log n)

对行二分找可能行，再对列二分。总 O(log m + log n)。

---

### 🟢 第二层：排除法 O(m+n)

```cpp
int i = 0, j = n - 1;
while (i < m && j >= 0) {
    int cur = matrix[i][j];
    if (cur == target) return true;
    else if (cur < target) ++i;
    else --j;
}
return false;
```

**图解：**

```
起点 (0,3)=7 → 7<11 → i++ → (1,3)
(1,3)=20 → 20>11 → j-- → (1,2)
(1,2)=16 → 16>11 → j-- → (1,1)
(1,1)=11 → 11==11 → 找到 ✅
```

**复杂度：**
- 时间：O(m+n)
- 空间：O(1)

---

## 破题总结

> **从右上角排除：target 小于当前值则向左（排除一列），大于当前值则向下（排除一行）——每次排除一行或一列，总步数不超过 m+n。**
