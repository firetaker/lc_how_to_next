# 合并区间（LeetCode #56）

## 题目描述

> 合并所有重叠的区间。

**示例：**
- 输入：`[[1,3],[2,6],[8,10],[15,18]]`
- 输出：`[[1,6],[8,10],[15,18]]`

---

## 梯度分析

### 🔵 第一层：按左端点排序后逐个合并

**核心洞察：** 按左端点排序后，所有可能重叠的区间都会相邻。

```python
intervals.sort(key=lambda x: x[0])
result = [intervals[0]]
for l, r in intervals[1:]:
    if l <= result[-1][1]:        # 有重叠
        result[-1][1] = max(result[-1][1], r)
    else:                          # 无重叠，开新区间
        result.append([l, r])
```

**图解 `[[1,3],[2,6],[8,10],[15,18]]`：**

```
排序后: [1,3], [2,6], [8,10], [15,18]

遍历:
[1,3] → result = [[1,3]]
[2,6] → 2 <= 3，重叠 → 合并为 [1, max(3,6)=6]
[8,10] → 8 > 6，无重叠 → 新增 [8,10]
[15,18] → 15 > 10，无重叠 → 新增 [15,18]

结果: [[1,6], [8,10], [15,18]]
```

**复杂度：**
- 时间：O(n log n)（排序）
- 空间：O(1)（不计结果空间）

---

## 破题总结

> **按左端点排序后，只需比较当前区间的左端点与已合并区间最后一个的右端点——有重叠则合并右端点，无重叠则开新区间，一遍遍历搞定。**
