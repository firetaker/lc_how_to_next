# 搜索二维矩阵（LeetCode #74）

## 题目描述

> 给定 m×n 矩阵，每行升序，每行第一个元素大于上一行最后一个元素。判断 target 是否存在。

**示例：**
- 输入：`[[1,3,5,7],[10,11,16,20],[23,30,34,60]], target=3`
- 输出：`True`

---

## 梯度分析

### 🔵 第一层：二维二分 O(log m + log n)

对行二分找到可能行，再对列二分。总 O(log m + log n)。

---

### 🟢 第二层：排除法 O(m+n)（最优）

**核心洞察：** 从右上角出发——
- `target == cur`：找到 ✅
- `target < cur`：当前列所有元素都太大 → **排除当前列**，j--
- `target > cur`：当前行所有元素都太小 → **排除当前行**，i++

```python
i, j = 0, n - 1  # 从右上角开始
while i < m and j >= 0:
    cur = matrix[i][j]
    if cur == target: return True
    elif cur < target: i += 1  # 往下走
    else: j -= 1               # 往左走
return False
```

**图解：**

```
matrix:
[1, 3, 5, 7]
[10,11,16,20]
[23,30,34,60]
target = 11

起点 (0,3)=7, 7<11 → i++ → (1,3)
(1,3)=20, 20>11 → j-- → (1,2)
(1,2)=16, 16>11 → j-- → (1,1)
(1,1)=11, 11==11 → 找到 ✅
```

**复杂度：**
- 时间：O(m+n)，每步至少排除一行或一列
- 空间：O(1)

---

## 破题总结

> **从右上角排除：target 小于当前值则向左（排除一列），target 大于当前值则向下（排除一行）——每次排除一行或一列，总步数不超过 m+n。**
