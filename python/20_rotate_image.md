# 旋转图像（LeetCode #48）

## 题目描述

> 顺时针旋转 n×n 矩阵 90 度，原地修改。

**示例：**
- 输入：`|1,2,3|` → 输出：`|7,4,1|`
         `|4,5,6|`         `|8,5,2|`
         `|7,8,9|`         `|9,6,3|`

---

## 梯度分析

### 🔵 第一层：另存旋转后坐标 O(n²)

手动计算旋转后坐标 `(i,j) → (j, n-1-i)`，另开数组保存。

---

### 🟢 第二层：两次翻转 O(1)

**核心洞察：** 顺时针 90° 旋转 = **沿主对角线翻转** + **沿水平中线翻转**

```
沿主对角线翻转: matrix[i][j] → matrix[j][i]
沿水平中线翻转: matrix[i][j] → matrix[i][n-1-j]

两步合成: 顺时针90度旋转
```

**验证：**

```
原始:
[1,2,3]
[4,5,6]
[7,8,9]

Step 1: 沿主对角线翻转 (转置)
[1,4,7]
[2,5,8]
[3,6,9]

Step 2: 沿水平中线翻转 (每行reverse)
[7,4,1]  ← row[0] reverse
[8,5,2]  ← row[1] reverse
[9,6,3]  ← row[2] reverse

结果: ✅ 顺时针旋转90度
```

```python
n = len(matrix)
# Step 1: 转置
for i in range(n):
    for j in range(i+1, n):
        matrix[i][j], matrix[j][i] = matrix[j][i], matrix[i][j]
# Step 2: 每行 reverse
for i in range(n):
    matrix[i].reverse()
```

**为什么等价于顺时针旋转？**
- 转置：沿主对角线镜像（右上三角→左下三角）
- 水平翻转：每行左右颠倒
- 两者复合 = 顺时针 90°

**复杂度：**
- 时间：O(n²)
- 空间：O(1)

---

## 破题总结

> **顺时针90度旋转 = 转置（主对角线翻转）+ 每行 reverse（水平翻转）。两步各 O(n²)，无需额外空间。**
