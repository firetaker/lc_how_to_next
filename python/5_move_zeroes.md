# 移动零（LeetCode #283）

## 题目描述

> 给定一个数组 `nums`，将所有 `0` 移动到数组末尾，同时**保持非零元素的相对顺序**。要求原地修改，O(1) 额外空间。

**示例：**
- 输入：`[0, 1, 0, 3, 12]`
- 输出：`[1, 3, 12, 0, 0]`

---

## 梯度分析

### 🔵 第一层：创建新数组（不符合 O(1) 要求）

最简单：遍历数组，收集非零元素，再补零：

```python
non_zero = [x for x in nums if x != 0]
return non_zero + [0] * (len(nums) - len(non_zero))
```

- 时间：O(n)
- 空间：O(n) ❌ 不符合 O(1) 要求

---

### 🟢 第二层：双指针原地交换

**核心思想：** 用两个指针在原数组上操作：
- `left`：左侧区域（`[0..left]`）都是非零元素
- `right`：当前遍历位置

每次遇到非零元素，就和 `left` 位置交换：

```python
left = 0
for right in range(len(nums)):
    if nums[right] != 0:
        nums[left], nums[right] = nums[right], nums[left]
        left += 1
```

**图解 `[0, 1, 0, 3, 12]`：**

| 步骤 | right | nums[right] | 交换后 nums | left |
|------|-------|-------------|-------------|------|
| 初始 | - | - | [0, 1, 0, 3, 12] | 0 |
| 1 | 0 | 0 | 跳过 | 0 |
| 2 | 1 | **1** | **[1**, 1, 0, 3, 12] → [1, **0**, 0, 3, 12] | 1 |
| 3 | 2 | 0 | 跳过 | 1 |
| 4 | 3 | **3** | [1, 0, **3**, 3, 12] → [1, 0, **3**, 12] | 2 |
| 5 | 4 | **12** | [1, 0, 3, **12**, 12] → [1, 0, 3, **12**, 0] | 3 |

最终 `[1, 3, 12, 0, 0]` ✅

**关键观察：** `left` 指针左边的元素始终是非零的，且相对顺序不变。

**复杂度：**
- 时间：O(n)，每个元素最多交换一次
- 空间：O(1)

---

## 破题总结

> **双指针中，`left` 维护「已处理非零元素」的边界，每当 `right` 遇到非零就与 `left` 交换并右移——类似「把非零元素往前挤」的视觉效果。**
